## 🚀 Featured Projects

### 📚 [Scholar Agent — AI Research Paper Agent](https://github.com/msaad732/ScholarAgent-AI-Research-Paper-Agent)
An **agentic RAG** research assistant where a **LangGraph ReAct agent** autonomously orchestrates 4 tools (knowledge-base search, arXiv fetch, cross-paper compare, metric extraction) to answer questions across a paper corpus — grounding every answer in ChromaDB-retrieved chunks with source citations.
* **Highlights:** Academic-aware chunking (splits PDFs by section), cross-encoder reranking, one-click *"review my paper"* critique, per-session isolated knowledge bases, token streaming.
* **Shipped production-style:** unit-tested with **pytest + GitHub Actions CI**, deployed on **Hugging Face (Docker)**.
* **Tech Stack:** Python, LangGraph, LangChain, ChromaDB, Groq (LLaMA-3), Streamlit, Docker.

### 📈 [Solana Meme Coin Tracker](https://github.com/msaad732/Meme-Coins-Tracker-System)
A real-time data pipeline that discovers newly launched Solana tokens, ingests Telegram signals, and analyzes wallet behaviors to detect market trends.
* **The Engine:** Custom **Playwright-based scraper** that bypasses anti-bot protections, capturing dynamic data every 30 seconds where standard APIs failed.
* **Tech Stack:** Python, Playwright, PostgreSQL, Streamlit, Railway.
* **Impact:** Automates 30+ metrics and builds a labeled dataset for future ML price-prediction models.

### 🌌 [NASA Space Apps Hackathon — ML Model & Chatbot](https://github.com/msaad732/nasa-hackathon-ml-model)
A Gradient Boosting classifier trained on **10,000+ near-earth-object records** (11 features) predicting hazardous classification at **79% accuracy**, integrated behind an NLP chatbot for conversational data queries.
* **Tech Stack:** Python, scikit-learn, Groq API, NLP.

### 🌱 [EcoLearn-AI — Multilingual Climate PWA](https://github.com/msaad732/ecolearn)
A 4-language AI tutoring app with a LLaMA-3 chatbot tutor, voice-driven Recycle Assistant, and gamified quizzes — pairing Groq Whisper (speech-to-text) with LLaMA-3 behind a Flask/FastAPI backend on Azure.
* **Tech Stack:** Python, FastAPI, LLaMA-3, Whisper, Azure.

### 🚑 [Medical Patrol (Full-Stack)](https://github.com/msaad732/medical-patrol-backend)
A healthcare alerting & dispatch system focused on real-time backend reliability and one-tap emergency reporting.
* **Tech Stack:** Python (Backend), HTML/CSS, REST APIs.
